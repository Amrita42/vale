|  NAME   |   TYPE   |                           DESCRIPTION                               |
|---------|----------|---------------------------------------------------------------------|
| `max`   | `int`    | The maximum amount of times `token` may appear in a given scope.    |
| `min`   | `int`    | The minimum amount of times `token` has to appear in a given scope. |
| `token` | `string` | The token of interest.                                              |
